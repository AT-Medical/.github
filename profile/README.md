Dieses Repository enthält organisationsweite GitHub-Konfigurationen der **AT Medical GmbH®**.  
Es stellt zentrale Standards, Vorlagen und Richtlinien bereit, die für alle Repositories innerhalb der AT-Medical-Organisation gelten.

Typische Inhalte sind unter anderem:

- globale GitHub-Konfigurationen  
- Issue- und Pull-Request-Templates  
- Sicherheits- und Community-Richtlinien  
- wiederverwendbare Workflow-Definitionen  
- Governance- und Qualitätsstandards  

Dieses Repository dient als zentrale Steuerungs- und Standardisierungsebene für die gesamte GitHub-Organisation.

This repository contains the organization-wide GitHub configuration of **AT Medical GmbH®**.  
It provides central standards, templates and policies used across all repositories within the AT-Medical organization.

Typical contents include:

- global GitHub configuration  
- issue and pull request templates  
- security and community policies  
- reusable workflow definitions  
- governance and quality standards  

This repository acts as the central control and standardization layer for the entire GitHub organization.
