## Scope

Deployment documentation in this directory describes how changes are deployed through the organization's GitHub-based CI/CD pipelines, including branch strategy, environment promotion, and release management.

For infrastructure- or product-specific deployment procedures, refer to the deployment documentation within the relevant repository.
