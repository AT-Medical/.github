# Governance Link

## Purpose

This document establishes the explicit relationship between the `.github` repository and the `github-governance` repository.

## Model

- `.github` = presentation and default behavior layer
- `github-governance` = policy, audit and enforcement layer

## Principle

All templates and defaults defined here must align with governance rules defined in `github-governance`.

## Enforcement

- deviations are corrected via governance processes
- audits are performed outside this repository

## Status

Compliant when both repositories are aligned and consistent.
